## Packages
framer-motion | Page transitions, scroll animations, and interactive elements
react-confetti | Final screen celebration effect
lucide-react | Elegant icons for the UI components
embla-carousel-react | Smooth, touch-friendly carousel for the memories section

## Notes
- Audio uses native HTML5 `<audio>` tag with a gentle placeholder URL, complying with browser autoplay policies by requiring a button click on the final screen.
- Memories will intelligently fallback to beautifully curated static data if the API is empty or fails, guaranteeing a flawless experience for the user's mother regardless of backend state.
- Embla carousel is used for smooth swipe gestures on mobile devices.
